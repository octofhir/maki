## Quick Example

```bash
# Install
cargo install maki

# Initialize configuration
maki init

# Format your FSH files
maki format input/fsh/**/*.fsh

# Lint your FSH files
maki lint **/*.fsh

# Auto-fix issues
maki lint --fix **/*.fsh
```

## What's FSH?

FHIR Shorthand (FSH) is a domain-specific language for defining FHIR Implementation Guides.
FSH Lint helps you catch errors early and maintain high-quality FHIR profiles.
