## Quick Example

```bash
# Install
cargo install maki

# Initialize a new IG project
maki init

# Build FSH to FHIR resources (SUSHI-compatible)
maki build --progress

# Lint your FSH files
maki lint input/fsh/

# Format your FSH files
maki fmt input/fsh/

# Convert FHIR back to FSH (GoFSH)
maki gofsh ./fsh-generated -o ./input/fsh
```

## What's FSH?

FHIR Shorthand (FSH) is a domain-specific language for defining FHIR Implementation Guides.
FSH Lint helps you catch errors early and maintain high-quality FHIR profiles.
